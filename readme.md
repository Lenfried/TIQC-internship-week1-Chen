CLI tool to parse a .csv file of student GPA scores and output the average GPA of students.

# Setup Instructions 
1. Clone the repository from GitHub
2. Ensure python3 is installed on your system (type python --version in terminal)
3. Place the .csv in the same directory (format for the .csv is Name,GPA)
4. Open a terminal in the folder and run the program with python gpa_cli.py
5. Input the file name of the .csv you would like to process

# Output
1. Number of rows (students) processed
2. Average GPA of students rounded to 2 decimal places
